About Arabic Explainer عن شرح البرمجة بالعربية

Our Mission مهمتنا

Arabic Explainer is a project created to help Arabic speakers understand and learn programming concepts in their native language. We believe that language should not be a barrier to learning code, and that everyone deserves access to quality programming education in a language they understand. شرح البرمجة بالعربية هو مشروع تم إنشاؤه لمساعدة المتحدثين بالعربية على فهم وتعلم مفاهيم البرمجة بلغتهم الأم. نؤمن بأن اللغة يجب ألا تكون عائقاً أمام تعلم البرمجة، وأن كل شخص يستحق الوصول إلى تعليم برمجي عالي الجودة بلغة يفهمها.

Why We Created This لماذا أنشأنا هذا المشروع

Many Arabic speakers face challenges when learning to code because most resources are in English. While learning English is valuable, we believe beginners should be able to grasp fundamental programming concepts in their native language first. This project aims to bridge that gap by providing clear, comprehensive explanations of code in Arabic. يواجه العديد من المتحدثين بالعربية تحديات عند تعلم البرمجة لأن معظم الموارد متوفرة بالإنجليزية. بينما تعلم اللغة الإنجليزية قيم، نؤمن بأن المبتدئين يجب أن يكونوا قادرين على فهم المفاهيم الأساسية للبرمجة بلغتهم الأم أولاً. يهدف هذا المشروع إلى سد هذه الفجوة من خلال تقديم شروحات واضحة وشاملة للكود بالعربية.

What We Offer ما نقدمه

  • Code Explanations: Detailed explanations of programming concepts in Arabic شروحات الكود: شروحات مفصلة لمفاهيم البرمجة بالعربية

  • Practical Examples: Real-world code examples with Arabic comments أمثلة عملية: أمثلة كود واقعية مع تعليقات بالعربية
  • Community Focused: Built for the Arab developer community تركيز على المجتمع: مبني لمجتمع المطورين العرب
