# Assets Directory

This directory contains static assets for the Arabic Explainer website:

- **Images**: Logo, icons, screenshots, and other images
- **Fonts**: Custom fonts (if needed)
- **Icons**: Favicon and other icon files
- **Media**: Videos or other media files

## Recommended Structure

```
assets/
├── images/
│   ├── logo.png
│   ├── hero-image.png
│   └── screenshots/
├── icons/
│   └── favicon.ico
├── fonts/
│   └── custom-fonts/
└── media/
    └── videos/
```

## Adding Assets

To add new assets:

1. Place files in the appropriate subdirectory
2. Reference them in HTML using relative paths: `assets/images/filename.png`
3. Optimize images for web (compress, resize as needed)
4. Use appropriate formats (WebP for modern browsers, PNG/JPG as fallback)
